신뢰구간 계산 전 필요한 모수 구하기 0. 성별에 따른 income 데이터프레임 생성

f_income = welfare.dropna(subset = "income") \
                  .query("sex == 'female'")['income']
m_income = welfare.dropna(subset = 'income') \
                  .query("sex == 'male'")['income']
  1. 성별 별 평균
f_bar = np.mean(f_income)
m_bar = np.mean(m_income)
  1. 표본표준편차 사용해서 모표분편차 추정
f_var = np.var(f_income, ddof=1)
m_var = np.var(m_income, ddof=1)

성별 별로 신뢰구간 계산하기

f_conv1 = norm.ppf(0.025, loc = f_bar, scale = np.sqrt(f_var))
f_conv2 = norm.ppf(0.975, loc = f_bar, scale = np.sqrt(f_var))
m_conv1 = norm.ppf(0.025, loc = m_bar, scale = np.sqrt(m_var))
m_conv2 = norm.ppf(0.975, loc = m_bar, scale = np.sqrt(m_var))
